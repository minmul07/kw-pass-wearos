# KW Pass for WearOS
Wear OS를 지원하는 광운대학교 도서관 qr코드 인증 어플리케이션

# 개발 중
## Phone
- QR 위젯
- 잠금화면에서 QR 확인
- And more
## WearOS
- 타일에서 QR 확인
- 시계 화면 컴플리케이션에서 QR 바로가기
- And more
